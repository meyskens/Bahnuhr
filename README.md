# Bahnuhr - DB watchface for Pebble

Informationen zur APP Bahnuhr, Abfahrt ursprünglich 13:12. Heute circa 120 Minuten später. Grund dafür ist ein neues Pebble-Watchface für die Bahnuhr.


## Base

This watchface is based on [ModernSwiss](https://github.com/meyskens/ModernSwiss) which is a upgrade of original work by Johnny Lim's Mondaine watchface, forked by Leo Lin then updated by us and dutchified.
